JavaScript "underfined" ve
"Not defined" mahiyyet ve ferqleri

Javascriptde underfined deyilen bir anlayish var.
underfined dediyimizde : variable var amma duzgun qeyd olunmayib
Meselen:

< script>
var Galatasaray;
console.log(Galatasaray) //underfined
console.log( "Galatasaray" )
< /script>

Not defined:
Eger teyin olunmayan varible istifade etmek istesek
o zaman bize " is not defined" verecek.
Meselen:

< script>
a; //is not defined
a=5 //duzgundur
function function_name() {
console.log(a)
}
function_name();
< /script>
