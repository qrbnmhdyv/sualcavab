NaN Nedir?

NaN Javascript-də sirli bir xüsusiyyətdir.
NaN bir nömrə deyil, bu əmlak sirli bir şeydir, "NaN == NaN return false", yəni iki NaN bərabər deyil.
Ona görə də onları sadəcə olaraq tənzimləməklə yoxlaya bilmərik. Xoşbəxtlikdən javascript,
dəyərin NaN olub olmadığını yoxlamaq üçün ISNaN funksiyasını təmin edir.
Bele desek NaN bir dəyərin qeyri-qanuni bir nömrə olub-olmadığını müəyyən edən bir JavaScript funksiyasıdır (Not-a-Number).
Dəyər NaN-ə bərabər olduqda funksiyanı qaytarır. Əks təqdirdə yalanı qaytarır.

< script >
var first = 0;
var second = 5;

var sum = first + second;

if (isNaN(sum)){
console.log("Deyer NaNdir");
}

else{
console.log("Deyer NaN deyil");
}
< /script>
